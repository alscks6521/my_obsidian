**BLE 사이클**  

1. 깨어나기 - 필요할 때만 활성화
2. 데이터 전송 - 작은 패킷을 빠르게 전송
3. 다시 잠들기 - 긴 수면 시간으로 전력 최소화


---  
**통신 규칙**  

| 구성 요소 | 역할                     |
| ----- | ---------------------- |
| GAP   | 기기 역할 및 연결 관리          |
| GATT  | 데이터 교환 형식 정의           |
| ATT   | 서비스 발견 및 데이터 읽기/쓰기     |
| SM    | 암호화 및 사용자 보안 처리        |
| L2CAP | 데이터 패킷 관리 (흐름 및 오류 제어) |
